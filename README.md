# Medical Image Analysis Practices

This repository contains academic practices related to computer vision and deep learning, with a focus on image segmentation, reconstruction and anomaly detection.

## Topics

- Image preprocessing
- Edge detection and thresholding
- Morphological operations
- Watershed segmentation
- CNNs for image classification
- U-Net for image segmentation
- Autoencoders for image reconstruction
- Anomaly detection using reconstruction error

## Technologies

- Python
- NumPy
- OpenCV
- Scikit-image
- TensorFlow / Keras
- PyTorch
- Jupyter Notebook

## Purpose

The goal of this repository is to showcase practical knowledge in computer vision and deep learning techniques that can be applied to medical image analysis, clinical decision support and biomedical data processing.
